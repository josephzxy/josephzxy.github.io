HBase shell 无法使用

Posted on 2018-05-06 | Edited on 2018-12-30 | In Distributed Computing

报错信息

原因

出现这种问题的原因是因为多台节点的时间不同步，导致节点之间的连接时间超时

解决方案

  1. 通过 ntpdate 0.cn.pool.ntp.org 命令使各个节点的时间跟网络时间同步
  2. 通过data -s命令手动调整节点时间
