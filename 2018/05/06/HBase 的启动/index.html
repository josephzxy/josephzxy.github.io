HBase 的启动

start-hbase.sh
这个会启动整个hbase，包括master, regionserver, zookeeper

如果要单独启动master和regionserver，可参考以下方式

在regionServer上 hbase-daemon.sh start regionserver

在master上执行：hbase-daemon.sh start master
